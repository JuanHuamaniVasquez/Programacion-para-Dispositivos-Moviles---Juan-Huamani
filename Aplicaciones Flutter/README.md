
# Aplicación Personal en Flutter

## Descripción
Este proyecto es una práctica de interfaces estáticas en Flutter.  
Consiste en tres pantallas visuales diseñadas con widgets básicos como `Text`, `Image`, `Column`, `Row`, `Padding` y `Container`.  
No se incluye lógica ni navegación; su objetivo es aprender a construir interfaces limpias y bien estructuradas.

---

## Estructura del proyecto
```

lib/
├─ main.dart
├─ pantalla_inicio.dart
├─ pantalla_perfil.dart
└─ pantalla_hobbies.dart

````

Cada archivo representa una pantalla distinta:
-  **Pantalla de Inicio:** mensaje de bienvenida y botón “Ver mi perfil”.
-  **Pantalla de Perfil:** imagen, nombre, descripción y contacto.
-  **Pantalla de Hobbies:** lista de actividades favoritas con íconos.

---

## Ejecución del proyecto

1. **Clonar con GitHub Desktop**
   - Abre *GitHub Desktop* → **File → Clone repository...**  
   - Pega la URL del repositorio, por ejemplo:  
     ```
     https://github.com/tu_usuario/actividad_bnavigationbar.git](https://github.com/JuanHuamaniVasquez/Programacion-para-Dispositivos-Moviles---Juan-Huamani.git
     ```
   - Elige una carpeta local y presiona **Clone**.

2. **Abrir en Android Studio**
   - Abre *Android Studio* → **Open an existing project**  
   - Selecciona la carpeta del proyecto (donde está el archivo `pubspec.yaml`).  
   - Espera a que Flutter sincronice los archivos automáticamente.

3. **Ejecutar la aplicación**
   - Presiona Run en Android Studio
